# ReMiddy - desktop app built with Electron

> Your own virtual assistant (and BFF) 🤍

ReMiddy is a Y2K-aesthetic desktop reminder app built with **React + Electron**. It sits on your desktop and pops up friendly, customizable reminders so you never forget to drink water, take a break, or anything else you need a nudge for.

---

## Features

- **Custom reminder intervals** - choose from 20, 40, 60, or 120 minutes, or set any custom value
- **Native desktop notifications** - reminders pop up as on-screen toasts that auto-dismiss
- **Manage your reminders** - add and delete reminders at any time
- **Y2K-inspired UI** - checkerboard vibes, pixel fonts

---

## 🛠 Tech Stack

| Layer         | Tech     |
| ------------- | -------- |
| UI            | React    |
| Desktop shell | Electron |
| Bundler       | Vite     |
| Styling       | CSS      |

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/skarolinax/reminder-app.git
cd my-app

# Install dependencies
npm install
```

### How to run

```bash
npm run start
```

Outputs a distributable desktop app for your device.
